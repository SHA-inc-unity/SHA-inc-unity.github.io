# SHA-inc-unity.github.io

Static GitHub Pages portfolio for `shainc.me`.

## Routes

- `/` - English portfolio
- `/resume/` - English main resume
- `/resume/backend/` - English backend resume
- `/resume/mobile/` - English mobile resume
- `/resume/ml/` - English ML / neural models resume
- `/ru/` - Russian portfolio
- `/ru/resume/` - Russian main resume
- `/ru/resume/backend/` - Russian backend resume
- `/ru/resume/mobile/` - Russian mobile resume
- `/ru/resume/ml/` - Russian ML / neural models resume

Legacy English paths under `/en/` redirect to the root English pages.
